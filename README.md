## 天津大学OJ题解  

目前状态：随2025级程序设计原理课程（05099）同步更新，偶尔增加OJ上其他题目。  

欢迎建议和讨论。  

### 题目来源：[天津大学OJ（TJUOJ）](https://acm.tju.edu.cn)  
如果你不是天津大学学生，可以注册账号并选择"账号登录"。  


题号查看方式：
- 一般为题目url的后部，例如：[`https://acm.tju.edu.cn/problem/A1000`](https://acm.tju.edu.cn/problem/A1000)链接到的题目题号为`A1000`。  
- 也可以直接在`题目`-->`题目列表`左侧搜索框中搜索题号。  

### 请注意：  

- 非官方题解，可能存在错误，仅供参考。  
- `#include<bits/stdc++.h>`是万能头文件，并非好的代码习惯。  
- 一般情况下，主要代码写在`solve()`函数中。
